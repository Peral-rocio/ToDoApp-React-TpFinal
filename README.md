> Mi primer proyecto desarrollado con React.js

---

## 📋 Descripción

**BambooTask** es una aplicación de gestión de tareas desarrollada con React.js como proyecto final. Permite crear, completar y eliminar tareas, con persistencia de datos mediante localStorage.

El proyecto también incorpora prácticas de desarrollo profesional como el uso de GitHub con ramas por funcionalidad, Pull Requests y flujo de integración hacia `develop` y luego a `master`.

---

## ✨ Funcionalidades

- ➕ Agregar nuevas tareas mediante un input con ícono
- ✔️ Marcar tareas como completadas (con tachado visual)
- 🗑️ Eliminar tareas del estado de la aplicación
- 🔍 Filtrar tareas por: **Todas / Completadas / Incompletas**
- 💾 Persistencia de datos con **localStorage**
- 🌙 Modo oscuro / claro
- 🔐 Login con cierre de sesión (con confirmación via SweetAlert2)
- 📅 Calendario integrado
- 📱 Diseño responsive
